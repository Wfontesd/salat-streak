# Salat Streak

Simple app mobile offline pour aider à garder une régularité dans les prières quotidiennes.

## Pourquoi cette idée
- pas de pub
- pas d'infra
- valeur claire
- monétisation simple via achat unique/premium plus tard
- compatible avec une contrainte "pas de haram"

## Proposition de valeur
- suivre ses prières du jour
- maintenir une streak hebdomadaire
- voir des stats simples
- recevoir une motivation douce
- tout reste local sur l'appareil

## Cible réaliste pour 100$/mois
- achat unique à 4.99$
- objectif ~25 ventes/mois
- ou abonnement optionnel léger plus tard si vraiment utile, mais l'achat unique suffit pour démarrer

## MVP
- suivi Fajr / Dhuhr / Asr / Maghrib / Isha
- coche journalière
- streak actuelle et meilleure streak
- historique 30 jours
- citations motivantes locales
- export/import JSON local plus tard
